Roblox Projects

A list of my Roblox Projects, past and present.

Probably one of the biggest projects I've lead, Parkour Race is a game where you race other people to finish the parkour first. I'm working on this with my friend (together we're known as ByteCode Studios).

Button of Doom


An old game I made in Roblox. It was a game where you had to press a button & and it would choose a random event to happen. Really fun to play, but I don't think it's worth it anymore.

Building System Test


As it's name suggests, this is a test of a Roblox building system that I made.

Just Climb


Well, just climb, as simple as that. A game I made in under 1 hour to prove that I can do it to my friends. Never really finished it, but I'm pretty sure it's still playable.

Terrain Generation


A project where I learnt the basics of how to use Perlin Noise to generate terrain. Nothing more, nothing less.


Node.js Projects

A list of my Node.js Projects, past and present. Most of them are Electron apps & Discord bots.

Byte


Utility & moderation Discord bot I made for the official ByteCode Studios Discord Server. Equipped with a plethora of commands (including slash commands now too!), including moderation, utility, and more.

YT Converter


A Node.js project that converts a YouTube video into an MP3/MP4 file. I made this project to learn how to use the ytdl-core package with Node.js & Electron. This also ended up being a learning experience for Fluent FFMPEG.

Not available to be "checked out" for obvious reasons.

MineBot


A Minecraft bot that does PvP. I made this project to learn how to use the Mineflayer package with Node.js.

Not available to be "checked out" for obvious reasons.
